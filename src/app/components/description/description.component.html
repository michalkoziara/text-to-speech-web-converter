Text-to-Speech converter

Convert text files or text to speech using Google Text-to-Speech service.

The only need is to pass your own developer API key from Google. Google provides free service for up to 4 million characters monthly however it does not provide user interface and HTTP requests are limited to 5000 characters.

This tool is designed to use your own API key to convert large text files or text that contains more than 5000 characters to MP3 files.
